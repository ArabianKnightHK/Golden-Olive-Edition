##### This mod tries to display realistic elements with higher graphics as well as pleasant songs with attractive gameplay, currently the mod is in the upgrade stage and in some parts it uses textures inspired by the SHC-DE version and sounds of SHC2. If for any reason you do not want or cannot play the latest version of Stronghold Crusader, this mod is a good alternative to it and even is better than it in some materials, you can also inform your comments and ideas for better development and if you wish.

#### Notes :

#####  To prevent interfrence and ensure proper functioning of this mod pack, place it in higher or newer plugins.

#####  This project is optimized according to the settings of the Graphic API Replacer module.
