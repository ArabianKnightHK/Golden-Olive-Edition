##### در این پروژه تلاش شده است تا حالت واقع گرایانه با گرافیک بالا به نمایش گذاشته شود، در حال حاضر این ماد در حال توسعه می باشد و در برخی قسمت ها از بافت های نسخه قطعی جنگ های صلیبی استفاده شده است و صدا های بازی هم در برخی قسمت ها مربوط به جنگ های صلیبی دو استو.

#### نکات مهم :

#####  برای جلوگیری از تداخل با سایر ماد ها این ماد را در افزونه بالاتر یا جدیدتر قرار دهید تا همه محتویات به درستی اجرا شوند.

#####  این پروژه برای تنظیمات ماژول Graphic API Replacer بهینه شده است ولی بدون استفاده از ماژول هم فعلا به درستی کار می کند اما استفاده از این ماژول توصیه می شود.

#####  هرگونه استفاده تجاری از این محصول ممنوع بوده و صرفاً برای استفاده شخصی می باشد.
